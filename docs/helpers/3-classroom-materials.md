## What's left for you (~1.5 hours total)

Two visual exports. Open the Markdown files, copy the content into a real design tool, and save the result back to the repo.

### 1. Debrief slide deck → PDF (~1 hour)

1. Open [Google Slides](https://slides.google.com) → **+ Blank**
2. Title it **"Star Bite Diner — Debrief"**
3. Open `/docs/teacher/02-debrief-outline.md` side-by-side
4. Build 8 slides matching the outline. The text is all written — your job is the **visual layout**:
   - Pick a theme. Suggested palette: dark navy (#0F1133), warm yellow (#FFD166), pink accent (#FF7EB3) — matches the game's UI.
   - For each slide, paste the title + bullets. Add icons/emoji where the outline calls for visuals.
   - Speaker notes go in the slide notes section.
   - For slides marked "screenshot of game UI," ask Hara to send you a screenshot — or just take one yourself by opening `https://starbite.onrender.com` (once Shreya deploys).
5. **File → Download → PDF Document**
6. Save as `02-debrief-deck.pdf` in `/docs/teacher/`

### 2. Rules card → PDF (~30 min)

1. Open [Google Docs](https://docs.google.com) → **Blank**
2. **File → Page setup → Landscape** + US Letter
3. Open `/docs/teacher/03-rules-card.md` side-by-side

5. **File → Print preview** to make sure it fits one page
6. **File → Download → PDF Document**
7. Save as `03-rules-card.pdf` in `/docs/teacher/`

## Tone guide (from the original spec — still applies)

- Warm, friendly, slightly silly. For 9-to-13-year-olds, not a corporate training.
- No condescension — speak to students like capable people.
- Pre-game script + rules card: **no AI/ML jargon**. Save it for the debrief.
- Debrief deck: connect gameplay moments to real-world AI students actually use. Frame mistakes as data problems, not "stupid AI."
- The bot is **never** the enemy. It's the team's trainee.
- Saboteurs are goofy mischief, not menace.

---

## A note on visual style

The debrief deck is the actual graded artifact. Don't agonize on theme — Google Slides' "Simple Light" or "Simple Dark" theme is fine. Spend your time making sure each slide reads at a glance and there's space for student discussion.
